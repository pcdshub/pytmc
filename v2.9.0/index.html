Welcome to pytmc’s documentation!¶

Pytmc helps developers automatically generate ADS based Epics records files from Beckhoff’s TwinCAT3 projects.

Pytmc is developed with these guidelines in mind.

The source code is hosted on github.

Issues and requests can be posted on the github issue tracker. Use the bug tag for issues with intended features and the enhancement tag can be used for feature requests. The question tag will be treated like an ‘enhancements’ tag for the documentation but regular questions can be posted there as well.
