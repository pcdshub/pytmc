Command-line Usage
==================

Using pytmc
-----------
Once pytmc has been installed in a virtual environment the ``pytmc`` command
can be called from the command line to generate .db files and more.


pytmc db
--------

.. argparse::
   :module: pytmc.bin.db
   :func: build_arg_parser
   :prog: pytmc db
